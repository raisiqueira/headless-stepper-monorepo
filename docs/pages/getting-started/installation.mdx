# Installation

You can install using [NPM](https://www.npmjs.com/), [Yarn](https://yarnpkg.com/en/docs/install) or [PNPM](https://pnpm.io/).

Headless Stepper is compatible with **React 16.8.0** and up

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>
  ```bash
  npm run dev
  ```
  </Tab>
  <Tab>
  ```bash
  yarn dev
  ```
</Tab>

  <Tab>
  ```bash
  pnpm dev
  ```
  </Tab>
</Tabs>
